## 💎 Key Features

### 1. **Antigravity Smooth-Scroll Experience**
Integrated with the **Lenis** inertia engine, tuned for "Snappy-Smooth" response (0.8s duration). This delivers a high-friction, premium tactile scroll feel that eliminates traditional browser jitter.

### 2. **Extreme "Zero-Lag" Particle Engine**
A custom **HTML5 Canvas** constellation engine running at 60FPS. Refactored with an **O(n) optimization layer** to allow hundreds of interactive particles to react to mouse coordinates without impacting the main-thread or GPU memory.

### 3. **Ecosystem Galaxy (Orbital Visualization)**
A cinematic orbital map of the entire SNISHOP infrastructure. Applications are rendered as synchronized planetary nodes across 4 orbital rings, utilizing **3D hardware-accelerated CSS transforms**.

### 4. **Super-Intelligent Local AI Brain**
A fully localized, **offline AI Assistant** with a proprietary **NLP Keyword Scoring Engine**. It features a modular multi-file knowledge base covering profile data, corporate vision (SNISHOP), and 20+ specialized app domains—no API keys or network latency required.

### 5. **Command Palette (⌘ + K)**
A global powerhouse gateway for instant ecosystem navigation. Execute commands, search for apps, and jump between sections with lightning speed via a sleek, keyboard-centric interface.

### 6. **Magnetic Interaction Engine**
Custom physics-driven hooks that grant UI elements "Cursor Gravity." Buttons and flagship cards possess a subtle lean-and-slide interaction, creating a high-fidelity tactile bond with the user.

## 🧠 Performance & Architecture

The system is governed by the **"Singularity Standard"**—a protocol ensuring absolute visual perfection without sacrificing performance.

*   **Spatial Partitioning**: Particle connection checks are optimized to prevent O(n²) bottlenecks.
*   **Compositor-Only Operations**: Animations target `transform` and `opacity` to avoid layout thrashing and repaints.
*   **Fluid Typography**: Uses `clamp()` functions globally for seamless 320px-to-8K scaling.
*   **Asset Buffering**: A custom SNAP-UP `PreLoader` masks the DOM painting phase with an easing 0-100% sequence.

## 📦 Getting Started

### Prerequisites
- Node.js (v18.0 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/SIRAJcrypto11/sirajnurihrom.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```

The project is optimized for **Vercel** with a custom `vercel.json` to handle client-side routing for Single Page Application (SPA) functionality.

```json
{
  "rewrites": [{ "source": "/(.*)", "destination": "/index.html" }]
}
```
